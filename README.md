# Nuxt 3 + Pinia + @pharen/ui Example

This project demonstrates a minimal Nuxt 3 frontend using Pinia for state management and the `@pharen/ui` UI library.

## Setup

1.  Install dependencies:

    ```bash
    npm install
    ```

2.  Run the development server:

    ```bash
    npm run dev
    ```

3.  Open your browser and navigate to `http://localhost:3000/calculator` to see the calculator page.

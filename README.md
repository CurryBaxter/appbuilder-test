# Nuxt 3 + Pharen UI Example

This project demonstrates a simple Nuxt 3 frontend using the Pharen UI library.

## Setup

1.  Install dependencies:

    ```bash
    cd frontend
    npm install
    ```

## Development

1.  Run the development server:

    ```bash
    npm run dev
    ```

    Open your browser and navigate to `http://localhost:3000`.

## Calculator

The project includes a basic calculator component located at `/calculator`.

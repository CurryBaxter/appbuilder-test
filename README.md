# Nuxt 3 Calculator App

This is a simple calculator app built with Nuxt 3 and the @pharen/ui component library.

## Instructions

1.  **Install dependencies:**

    ```bash
    cd frontend
    npm install
    ```

2.  **Run the development server:**

    ```bash
    npm run dev
    ```

3.  **Open the app in your browser:**

    Go to `http://localhost:3000/calculator`

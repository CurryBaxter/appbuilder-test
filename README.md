# Calculator App

This project is a simple calculator app built with Nuxt 3 and the Pharen UI library.

## Setup

1.  Clone the repository.
2.  Navigate to the `frontend` directory.
3.  Install dependencies: `npm install`
4.  Run the development server: `npm run dev`

## Usage

Open your browser and go to `http://localhost:3000/calculator` to use the calculator.

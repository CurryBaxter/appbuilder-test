# Nuxt 3 + Pinia + @pharen/ui Calculator

A simple calculator application built with Nuxt 3, Pinia for state management, and @pharen/ui for the UI components.

## Installation

1.  Clone the repository.
2.  Navigate to the `frontend` directory:
    ```bash
    cd frontend
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

## Development

To run the application in development mode:

```bash
npm run dev
```

## Build

To build the application for production:

```bash
npm run build
```

## Usage

Navigate to `/calculator` to use the calculator.

# Calculator App

## Frontend

Built with Nuxt 3 and @pharen/ui.

### Setup

```bash
cd frontend
npm install
npm run dev
```

## Backend

Built with Django and DRF.

### Setup

```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## Infrastructure

Docker and Kubernetes configurations are provided under the `infra` directory.
